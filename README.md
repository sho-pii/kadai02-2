## 紹介と使い方

  - 「晩ごはんルーレット」
  - 料理名とイラストをランダムで表示するアプリです。
  - 気分に合わせて「こってり」と「あっさり」のどちらかのボタンを押すとルーレットが開始されます。
  - 「ストップ」を押すと、イラストが止まり、料理名が表示されます。
  - 「もどる」を押すと、はじめの画面に戻ります。
